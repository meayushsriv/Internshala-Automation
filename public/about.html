About Internshala Auto-Apply Bot

The Internshala Auto-Apply Bot is a smart and efficient tool developed to simplify and automate the internship application process on Internshala. Designed for students and job seekers, this tool removes the repetitive effort of filling out applications by auto-submitting them intelligently based on the user's profile and resume.

By leveraging automation through Puppeteer and AI capabilities via the Gemini API from Google, the bot can log in, navigate internship listings, fill out response fields, and applyâ€”without human intervention. All it needs is your Internshala login and your resume in PDF format.

Built using a modern tech stack, including Node.js, Express, Bootstrap, and Puppeteer, this tool represents the fusion of backend automation and intelligent frontend interactions. The Gemini API helps craft contextual and relevant answers to custom application questions, saving time while improving the quality of applications.

You can find the complete source code and documentation on the GitHub repository linked below. This tool is an open-source project created with passion and the desire to simplify real-world problems using technology.
